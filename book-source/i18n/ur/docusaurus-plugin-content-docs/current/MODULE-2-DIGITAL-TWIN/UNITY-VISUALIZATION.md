# ہیومنائڈ روبوٹکس کے لیے یونیٹی ویزولائزیشن

جدید ہیومنائڈ روبوٹکس ڈویلپمنٹ کے لیے روبوٹ رویہ، سینسر ڈیٹا، اور ماحولیاتی انٹرایکشنز کو سمجھنے کے لیے پیچیدہ ویزولائزیشن کیپیبلٹیز کی ضرورت ہوتی ہے۔ اپنے طاقتور گرافکس انجن اور فزکس سیمولیشن کے ساتھ، یونیٹی روبوٹس کے لیے اممرسو 3D ویزولائزیشن بنانے کے لیے ایک بہترین پلیٹفرم فراہم کرتا ہے۔ یہ باب ریل ٹائم ویزولائزیشن کے لیے یونیٹی کو ROS2 کے ساتھ انٹیگریٹ کرنے کی تلاش کرتا ہے۔

## سیکھنے کے مقاصد

اس باب کے اختتام تک، آپ کر سکیں گے:
- ہیومنائڈ روبوٹکس کے لیے یونیٹی-ROS2 انٹیگریشن کی فن تعمیر اور کمپوننٹس کو سمجھنا
- ریل ٹائم کمیونیکیشن کے لیے یونیٹی روبوٹکس پیکیج اور ROS# بریج کو سیٹ اور کنفیگر کرنا
- ہیومنائڈ روبوٹ پوزز، جوائنٹ اینگلز، اور سینسر ڈیٹا کو ویزولائز کرنے کے لیے یونیٹی میں C# اسکرپٹس نافذ کرنا
- یونیٹی میں صحیح کائنیمیٹک چینز کے ساتھ حقیقی پسند ہیومنائڈ روبوٹ ماڈلز بنانا
- کیمروں، LiDAR، اور IMU ڈیٹا سمیت سینسر ویزولائزیشن کو انٹیگریٹ کرنا
- ROS2 نوڈز کے ساتھ رابطہ کرنے والی یونیٹی ویزولائزیشن ایپلیکیشنز ڈیپلائی کرنا

## تعارف

ویزولائزیشن ہیومنائڈ روبوٹکس ڈویلپمنٹ میں اہم کردار ادا کرتی ہے، جو محققین اور انجینئرز کو روبوٹ رویہ دیکھنے، کنٹرول الگورتھمز کی ڈیبگ کرنے، اور اسٹیک ہولڈرز کو کیپیبلٹیس ڈیمونسٹریٹ کرنے کی اجازت دیتی ہے۔ جبکہ Gazebo جیسے روایتی سیمولیشن ٹولز فنکشنل ویزولائزیشن فراہم کرتے ہیں، یونیٹی بہتر گرافکس معیار، انٹریکٹو کیپیبلٹیز، اور کراس پلیٹفارم ڈیپلے آپشنز پیش کرتا ہے۔

یونیٹی کی ریل ٹائم رینڈرنگ کیپیبلٹیز، اس کے وسیع اثاثے ایکوسسٹم، اور فزکس انجن کے ساتھ، یہ فوٹوریلزم ہیومنائڈ روبوٹ ویزولائزیشن بنانے کے لیے مثالی ہے۔ یونیٹی روبوٹکس پیکیج اور ROS# بریج یونیٹی اور ROS2 کے درمی بیکار کمیونیکیشن کو سہولت دیتے ہیں، جو روبوٹ اسٹیٹس، سینسر ڈیٹا، اور ماحولیاتی انٹرایکشنز کی ہم آہنگ ویزولائزیشن کی اجازت دیتے ہیں۔

### یونیٹی-ROS2 انٹیگریشن فن تعمیر

یونیٹی اور ROS2 کے درمی انٹیگریشن میں کئی اہم کمپوننٹس شامل ہیں جو بائیڈائریکشنل کمیونیکیشن اور ریل ٹائم سنکرونائزیشن کو ممکن بناتے ہیں:

**ROS# بریج**: ایک کمیونیکیشن لیئر جو یونیٹی کے اندر ROS2 کمیونیکیشن کو ممکن بناتا ہے۔ یہ ROS2 میسیج ٹائپس، پبلشر/سبسکرائبر سسٹم، TF ٹرانسفارم ہینڈلینگ، اور ROS2 سروس کالز کے لیے سپورٹ فراہم کرتا ہے۔

**یونیٹی روبوٹکس پیکیج (URP)**: روبوٹکس ایپلیکیشنز کے لیے یونیٹی کی کیپیبلٹیز کو بڑھاتا ہے، بشمول روبوٹکس لائبریری، سینسر سیمولیشن، TF ٹری مینیجمنٹ، اور فزکس انٹیگریشن۔

**کسٹم C# اسکرپٹس**: ایپلیکیشن-مخصوص لاجک جو ROS2 میسیجز کو ہینڈل کرتی ہیں۔

**ROS2 نوڈز**: روبوٹ اسٹیٹ اور سینسر ڈیٹا پبلش کرنے والے بیرونی نوڈز۔

### ROS# بریج کی بنیادیات

ROS# ( ROS Sharp پڑھا جاتا ہے) ایک یونیٹی پلگ ان ہے جو یونیٹی اور ROS2 کے درمی براہ راست کمیونیکیشن کو ممکن بناتا ہے۔ یہ فراہم کرتا ہے:

- **میسیج ٹائپس**: یونیٹی کے لیے پہلے سے متعدد ROS2 میسیج سٹرکچر
- **پبلشر/سبسکرائبر سسٹم**: ROS2 کمیونیکیشن کے لیےnative یونیٹی انٹرفیس
- **TF ٹرانسفارم ہینڈلینگ**: ROS2 اور یونیٹی کے درمی کوآرڈینیٹ سسٹم کنورژن
- **سروس کالز**: یونیٹی سے ROS2 سروس ریکویسٹس کے لیے سپورٹ

### یونیٹی روبوٹکس پیکیج کی خصوصیات

یونیٹی روبوٹکس پیکیج روبوٹکس ایپلیکیشنز کے لیے یونیٹی کی کیپیبلٹیز کو بڑھاتا ہے:

- **رobotics Library**: عام روبوٹکس ٹاسکس کے لیے پری-بلٹ کمپوننٹس
- **سینسر سیمولیشن**: وर्चوئل سینسرز جو ROS2 کو ڈیٹا پبلش کر سکتے ہیں
- **TF Tree Management**: خودکار ٹرانسفارم ہیرارکی مینیجمنٹ
- **فزکس انٹیگریشن**: بہتر کولیشن ڈیٹیکشن اور فزکس سیمولیشن
- **ROS2 میسیج جنریٹرز**: کسٹم میسیج ٹائپس بنانے کے لیے ٹولز

ہیومنائڈ روبوٹکس کے لیے، پیکیج پیچیدہ کائنیمیٹک چینز اور انورس کائنیمیٹکس سلورز کے لیے سپیشلائزڈ کمپوننٹس فراہم کرتا ہے۔

## یونیٹی میں ہیومنائڈ روبوٹ ماڈل بنانا

### 3D ماڈل کی تخلیق

یونیٹی میں ہیومنائڈ روبوٹ ماڈل بنانے کے لیے:

1. **ماڈل امپورٹ**: FBX یا OBJ فائل کے طور پر 3D ماڈل درآمد کریں
2. **کمپوننٹس شامل کریں**: Rigidbody، Collider، اور Joint کمپوننٹس شامل کریں
3. **جوائنٹ سیٹ اپ**: ہیومنائڈ کائنیمیٹکس کے لیے کنفیگریبل جوائنٹس کا استعمال کریں
4. **TF ہیرارکی**: صحیک ٹرانسفارم ساخت قائم کریں

### C# اسکرپٹنگ مثال

```csharp
using UnityEngine;
using ROSSharp.Core.Messages;
using ROSSharp.Core.Services;
using System.Collections.Generic;

public class HumanoidVisualizer : MonoBehaviour
{
    // ROS2 بریج کنکشن
    private ROSNodeSubscriber subscriber;

    // روبوٹ ماڈل کے جوائنٹس
    private Dictionary<string, Transform> jointTransforms;

    void Start()
    {
        InitializeJointMap();
        SetupROSSubscriber();
    }

    private void InitializeJointMap()
    {
        jointTransforms = new Dictionary<string, Transform>();
        // جوائنٹس کی میپنگ کی ترتیب
        FindJoint("left_hip_joint", transform);
        FindJoint("right_hip_joint", transform);
        // مزید جوائنٹس...
    }

    private void SetupROSSubscriber()
    {
        // ROS2 جوائنٹ اسٹیٹ ٹاپک کی سبسکرپشن
        subscriber = new ROSNodeSubscriber(
            "/joint_states",
            "sensor_msgs/JointState",
            OnJointStateReceived
        );
    }

    private void OnJointStateReceived(object message)
    {
        var jointState = (JointState)message;
        UpdateRobotPose(jointState);
    }

    private void UpdateRobotPose(JointState state)
    {
        for (int i = 0; i < state.name.Length; i++)
        {
            if (jointTransforms.ContainsKey(state.name[i]))
            {
                jointTransforms[state.name[i]].localRotation =
                    Quaternion.Euler(
                        (float)state.position[i] * Mathf.Rad2Deg, 0, 0);
            }
        }
    }
}
```

## سینسر ویزولائزیشن

### کیمرا فیڈ ویزولائزیشن

یونیٹی میں سیمولیٹڈ کیمرا فیڈ کو ویزولائز کرنے کے لیے:

1. رینڈر ٹارگیٹ کی ترتیب دیں
2. RawImage کمپوننٹ کے ذریعے ڈسپلے کریں
3. ROS2 Image میسیجز کو سبسکرائب کریں

### LiDAR پوائنٹ کلاؤڈ ویزولائزیشن

LiDAR ڈیٹا کو Particle System یا Point CloudRenderer کا استعمال کرتے ہوئے ویزولائز کیا جا سکتا ہے۔

## ڈیپلےمنٹ کے تحفظات

یونیٹی ویزولائزیشن ایپلیکیشنز کی ڈیپلےمنٹ کے لیے:

- **کراس پلیٹفارم سپورٹ**: Windows، Linux، macOS کے لیے تعمیر کریں
- **نیٹورک سیٹنگز**: ROS2 بریج کے لیے صحیح نیٹورک کنفیگریشن
- **پرفارمنس آپٹمائزیشن**: ریل ٹائم ویزولائزیشن کے لیے گرافکس سیٹنگز ایڈجسٹ کریں

## ٹراؤبل شوٹنگ

### کمیونیکیشن مسائل

- ROS2 بریج کنکشن نہیں ہو رہا: IP اور پورٹ سیٹنگز چیک کریں
- میسیجز موصول نہیں ہو رہیں: Topic ناموں کی تصدیق کریں

### ویزولائزیشن مسائل

- غلط روبوٹ پوز: TF ٹری ساخت چیک کریں
- لیگنگ: گرافکس سیٹنگز کم کریں

## باب کا خلاصہ

یہ باب ہیومنائڈ روبوٹکس کے لیے یونیٹی ویزولائزیشن کے اہم پہلوؤں کا احاطہ کرتا ہے۔ یونیٹی کی طاقتور گرافکس کیپیبلٹیز ROS2 کے ساتھ مل کر ہیومنائڈ روبوٹس کے لیے اممرسو اور معلوماتی ویزولائزیشن تجربہ فراہم کرتی ہیں جو ڈویلپمنٹ، ڈیبگنگ، اور پریزنٹیشن کے لیے قیمتی ہیں۔
